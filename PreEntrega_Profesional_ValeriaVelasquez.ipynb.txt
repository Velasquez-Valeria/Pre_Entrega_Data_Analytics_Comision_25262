## 🧹 4. Limpieza y preparación de datos <a name="limpieza"></a>
```python
# Verificar valores nulos
df.isnull().sum()

# Ejemplo de limpieza
df.drop_duplicates(inplace=True)
df['columna'] = df['columna'].fillna(df['columna'].median())
```
```markdown
**🧠 Decisiones tomadas:**
- Se eliminaron duplicados.  
- Se imputaron valores nulos con la mediana.  
- Se renombraron columnas para mayor claridad.  
```

## 📈 5. Análisis exploratorio de datos (EDA) <a name="eda"></a>
```python
# Ejemplo de gráfico
plt.figure(figsize=(8,5))
sns.histplot(df['columna_interes'], kde=True, color='teal')
plt.title('Distribución de la variable de interés 📊')
plt.xlabel('Variable')
plt.ylabel('Frecuencia')
plt.show()
```

```python
# Correlaciones
plt.figure(figsize=(10,6))
sns.heatmap(df.corr(), annot=True, cmap='coolwarm')
plt.title('Mapa de correlaciones 🔥')
plt.show()
```
```markdown
**🧐 Hallazgos clave del análisis:**
- Variable X muestra tendencia ascendente.  
- Variable Y correlaciona fuertemente con Z.  
- Segmento de edad 25–35 responde mejor a promociones.  
```
